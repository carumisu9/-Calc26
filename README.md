# Calc26

0~25 = A-Z

## Usage
```
$ python main.py "[A-Z]+ [+-*] [A-Z]+"
```

### Example
```
$ python main.py "A + B"
A: 0
B: 1
Answer_26: B
Answer_10: 1

$ python main.py "DE - CC"
DE: 82
CC: 54
Answer_26: BC
Answer_10: 28

$ python main.py "ZZ * FR"
ZZ: 675
FR: 147
Answer_26: FQUJ
Answer_10: 99225
```
